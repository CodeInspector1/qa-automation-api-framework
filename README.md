# ReqRes API Tests

Небольшой showcase-проект, демонстрирующий навыки написания API-тестов на Python с использованием современных инструментов - **pytest**, **pydantic** и **requests**.

## Зачем

Проект создан для моего публичного GitHub-профиля, чтобы показать практические примеры автоматизации API-тестирования.  
В основе - тестирование открытого сервиса [ReqRes API](https://reqres.in/).

## Что внутри

- Разделение на уровни:
  - `api_client.py` - универсальный клиент для HTTP-запросов  
  - `services/` - слой бизнес-логики (auth, users)
  - `schemas/` - pydantic-схемы запросов и ответов  
  - `tests/` - тестовые сценарии с фикстурами и параметризацией  

- Покрытие тестами:
  - Авторизация (успешные и неуспешные логины)
  - Получение пользователей
  - Создание, обновление и удаление пользователя
  - Проверка пагинации

## Используемые технологии

- Python 3.11
- Pytest
- Requests
- Pydantic

## Как запустить

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/<your-username>/reqres-api-project.git
   cd reqres-api-project

2. Установить зависимости:
   pip install -r requirements.txt

3. Запуск:
   pytest -v
